About Me

Hello! My name is Arthur, and I am a final-year Computer Science student passionate about software development, cloud computing, and cybersecurity. I have a strong interest in secure system design and enjoy working on projects that combine data security, efficiency, and scalability.

My Final Year Project is focused on developing a Farm Management System, a secure database driven application that helps farm owners manage essential operations such as animal health, feeding schedules, transactions, and medical history. This project combines modern web technologies, secure authentication, and data analytics to improve farm management efficiency.

In addition to programming, I actively explore cybersecurity concepts, particularly in areas such as network security, penetration testing, and secure software development.

My goal is to build secure, efficient, and scalable applications that have a meaningful impact. Whether it's through full-stack development, data security, or cloud computing.
